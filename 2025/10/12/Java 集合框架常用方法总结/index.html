Java 集合框架常用方法总结

2025-10-12 · 3 min · 920 words · Java 集合框架

🌿 一、Collection 接口（单列集合）

📘 共性方法（List、Set、Queue 都继承自 Collection）

方法 说明
add(E e) 添加元素
addAll(Collection c) 添加集合
remove(Object o) 删除指定元素
removeAll(Collection c) 删除多个元素
retainAll(Collection c) 取交集
clear() 清空集合
contains(Object o) 是否包含
isEmpty() 是否为空
size() 元素数量
iterator() 获取迭代器
forEach(Consumer action) 遍历（JDK8 Lambda）
stream() 转为流操作

1️⃣ List（有序，可重复）

🧱 常见实现类

  • ArrayList → 动态数组，随机访问快
  • LinkedList → 双向链表，插入删除快
  • Vector（已过时）
  • Stack（继承 Vector，已过时）
  • CopyOnWriteArrayList → 并发安全，读多写少

🔧 常用方法

方法 说明
get(int index) 获取指定下标元素
set(int index, E e) 替换元素
add(int index, E e) 指定位置插入
remove(int index) 删除指定位置元素
indexOf(Object o) / lastIndexOf(Object o) 查找索引
subList(int from, int to) 截取子列表
sort(Comparator c) 排序（JDK8）

1.2 Set（无序 / 有序，不重复）

🧱 常见实现类

  • HashSet → 基于 HashMap 实现，元素唯一
  • LinkedHashSet → 保持插入顺序
  • TreeSet → 基于 TreeMap，自动排序
  • CopyOnWriteArraySet → 并发安全集合
  • ConcurrentSkipListSet → 并发有序集合

🔧 常用方法

（继承自 Collection，无特有方法）
TreeSet 额外支持：

方法 说明
first() / last() 获取第一个/最后一个元素
headSet(E to) / tailSet(E from) 截取子集
subSet(E from, E to) 获取范围子集
comparator() 获取比较器

1.3 Queue / Deque（队列 / 双端队列）

🧱 常见实现类

  • PriorityQueue → 基于堆的优先队列
  • ArrayDeque → 高效双端队列
  • LinkedList → 也实现了 Deque
  • BlockingQueue（接口族）
    • ArrayBlockingQueue
    • LinkedBlockingQueue
    • PriorityBlockingQueue
    • DelayQueue
    • SynchronousQueue

🔧 常用方法

方法 说明
offer(E e) 入队（返回 boolean）
poll() 出队（返回元素或 null）
peek() 查看队首但不移除
element() 查看队首（若为空抛异常）
remove() 删除队首（若为空抛异常）

Deque 特有：

方法 说明
addFirst(E e) / addLast(E e) 头尾插入
removeFirst() / removeLast() 头尾删除
peekFirst() / peekLast() 查看头尾元素

2️⃣ Map 接口（双列集合：key-value）

📘 共性方法

方法 说明
put(K key, V value) 添加或覆盖键值对
get(Object key) 获取值
getOrDefault(Object key, V default) 默认值获取
remove(Object key) 删除键值对
containsKey(Object key) / containsValue(Object value) 是否包含
size() 键值对数量
clear() 清空
isEmpty() 是否为空
keySet() 所有键
values() 所有值
entrySet() 键值对集合
forEach(BiConsumer<K,V>) 遍历（Lambda）
putIfAbsent(K key, V value) 若不存在则添加
replace(K key, V value) 替换值
merge(K key, V value, BiFunction remapping) 合并规则（JDK8）

🧱 常见实现类

  • HashMap → 哈希表（JDK 1.8 起链表 + 红黑树）
  • LinkedHashMap → 保序哈希表
  • TreeMap → 红黑树结构，按 key 排序
  • Hashtable → 同步哈希表（过时）
  • ConcurrentHashMap → 高性能并发哈希表
  • ConcurrentSkipListMap → 并发有序 Map
  • WeakHashMap → 使用弱引用键
  • IdentityHashMap → 比较键用 ==
  • Properties → 键值均为字符串

3️⃣ 工具类

Collections（集合工具类）

方法 功能
sort(List) / sort(List, Comparator) 排序
reverse(List) 反转
shuffle(List) 随机打乱
max(Collection) / min(Collection) 求最大最小值
frequency(Collection, Object) 统计出现次数
synchronizedXXX(Collection) 获取线程安全集合包装
unmodifiableXXX(Collection) 获取只读集合包装
copy(List dest, List src) 复制集合内容

Arrays（数组工具类）

方法 功能
sort(array) 排序
binarySearch(array, key) 二分查找
equals(a1, a2) 比较内容
fill(array, val) 填充
copyOf(array, newLength) 扩容
asList(array) 数组转 List
stream(array) 数组转流

4️⃣ 特殊集合与扩展

  • EnumSet / EnumMap → 专为枚举类型设计，性能高
  • CopyOnWriteArrayList / Set → 并发安全（读多写少）
  • BlockingQueue 系列 → 支持线程阻塞
  • ConcurrentSkipListMap / Set → 并发有序结构
  • WeakHashMap → 键为弱引用，防止内存泄漏
  • IdentityHashMap → 使用 == 比较键
  • Properties → 键值均为字符串（配置文件常用）
